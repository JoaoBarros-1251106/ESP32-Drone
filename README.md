ESP32 Drone System

Projeto pessoal de desenvolvimento de um sistema completo de um drone quadricóptero baseado em ESP32.
Este projeto tem como objetivo desenvolver e compreender o funcionamento de um drone desde a base, incluindo o hardware, o software e a comunicação entre os diferentes componentes.

O sistema é composto por dois ESP32:
- Um ESP32 responsável pelo controlo do drone.
- Um ESP32 responsável pelo comando remoto, utilizando a aplicação ESPGO.

Todo o projeto será desenvolvido, testado e documentado ao longo da sua evolução.

Objetivos

- Construir um drone quadricóptero funcional.
- Desenvolver um comando remoto baseado em ESP32.
- Implementar comunicação sem fios entre o comando e o drone.
- Integrar sensores para estabilização e controlo do voo.
- Desenvolver o software de ambos os ESP32.
- Aprender e aplicar conceitos de sistemas embebidos, eletrónica e controlo em tempo real.
- Documentar todas as etapas do desenvolvimento.


Hardware:

Drone
- ESP32
- ESC 4 em 1
- 4 Motores Brushless 1104 7500kv
- Hélices
- MPU6050
- Bateria LiPo 2s 
- Conversor DC-DC

Comando
- ESP32
- ESPGO
- Joysticks
- Botões
- Bateria recarregável



Software

O desenvolvimento do software será realizado principalmente em C++ utilizando o Arduino IDE.

Entre as funcionalidades previstas encontram-se:
- Controlo dos motores por PWM.
- Leitura dos sensores.
- Comunicação sem fios entre os dois ESP32.
- Processamento dos comandos enviados pelo utilizador.
- Algoritmos de estabilização do voo.
- Gestão da alimentação e monitorização da bateria.

- Sistemas Embebidos
